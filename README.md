====== Идеи для кубиков =======
- Мотиватор, через определенное время увеличивает шанс критического урона у случайного кубика на доске
- Жадина, накапливает все пропущенные выстрелы других кубиков, и при накоплении критического количества, выстреливает их с максимальной скоростью.
- Инертность, начинает атаковать противника медленно увеличивая скорость атаки с каждым выстрелом. При переключении на нового противника, всё заново.


===== Идеи для Артефактов =====
- Двойная колода, например 1 раз в 30 волн можно поменять колоду
- Увеличивает базовый урон всех кубиков в колоде +1
- Артефакт позволяющий усиливат ькубики с помощью кубиков с 6-ью точками
